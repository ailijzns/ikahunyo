PROJECTS


A curated collection of my web projects showcasing HTML, CSS, and interactive design skills.


【 Pokédex-Style Data Viewer 】

╰┈➤ A web-based Pokémon database that lets users browse and filter by generation to view full Pokémon details — including name, image, types, evolution line, and more. It demonstrates dynamic data filtering and clean display of structured data using HTML, CSS and JavaScript.

【 Comeback Poster Layout 】

╰┈➤ A visually styled comeback announcement poster created using pure HTML and CSS. This project demonstrates layout design, text styling, image placement, and responsiveness without using any frameworks.

【 Interactive Starter Pokémon Showcase 】

╰┈➤ This project displays a Pokémon character with its animated GIF, name, type, and short description. It also includes fun “Slap” and “Clap” buttons for basic interaction, even though they do not trigger real actions. The website focuses on practicing layout, styling, and character presentation using HTML and CSS.

【 Winx Club Character Profile Viewer 】

╰┈➤ This project provides an overview of characters from the animated series Winx Club, including images and short descriptions of their magical role, background, relationships, and place in the group’s lineup. Its purpose is to present a clean, responsive, and informative profile that adapts to different devices for easy reference on both desktop and mobile screens.

【 Inside Out Character & Scene Viewer 】

╰┈➤ This project presents the characters, core emotions, and notable scenes from the movie Inside Out. It is designed to be clean, responsive, and visually engaging, allowing users to explore characters and scenes easily on both desktop and mobile devices.

【 Flexbox Layout Demo 】

╰┈➤ This project shows a demonstration of various CSS Flexbox properties — including flex direction, gap, justify‑content, align‑items, and flex‑grow settings — with several “box” elements that rearrange depending on the chosen layout. Its purpose is to provide a clean, responsive, and interactive reference that makes it easy to understand and experiment with Flexbox behaviors.

【 Pokémon Visual Gallery Viewer 】

╰┈➤ This project presents a clean, responsive gallery of Pokémon images. Designed for easy viewing on both desktop and mobile devices, it allows users to browse Pokémon visuals quickly and intuitively, focusing entirely on images rather than textual information.

【 Personal Portfolio Website 】

╰┈➤ A personal website showcasing my academic background, creative interests (blog entries on style, beauty, and fashion), and a gallery of my programming and design projects. The portfolio serves as a digital archive of my school‑assignments and creative work, providing visitors with a concise overview of my skills, studies, and contact information.

【 HTML & CSS Resume Recreation 】

╰┈➤ A digital recreation of a professional résumé using HTML and CSS. This project demonstrates web development skills by presenting educational background, skills, projects, and certifications in a structured, visually appealing format suitable for online portfolios or job applications.

【 Multiplication Table Web Page 】

╰┈➤ A web‑based page that displays a multiplication table, built using HTML and CSS. It demonstrates the ability to structure tabular data clearly, serving as an educational tool or interactive utility for learning multiplication.

【 Enrollment Form Page 】

╰┈➤ A web‑based page demonstrating the use of HTML elements for form inputs. The project highlights the proper structuring of lists and forms in a visually organized layout, serving as a learning reference for HTML basics.

【 Birth‑Month Calendar Page 】

╰┈➤ A simple HTML page displaying a March calendar, demonstrating the use of basic HTML elements and a moving text (marquee) effect. The project highlights content structuring and dynamic text presentation for learning purposes.
